# Demo Script

Use this when a judge has 5-8 minutes.

## Setup

```powershell
npm install
npm run db:generate
npm run db:push
npm run db:seed-users
npm run db:seed-registry -- --force
npm run db:seed-prompts
npm run dev
```

Password for all seeded accounts: `demo1234`.

## Walkthrough

1. Open `/demo`.
2. Open the completed seeded run and point out the `DEMO DATA` label.
3. Show provider matrix, validation contract, repo intelligence, evidence locker, skill graph, terminal proof, interview, AI challenge, and publish blockers.
4. Open `/profile/casey-candidate-skillproof-ai-demo` and show public-safe evidence only.
5. Sign in as `employer@skillproof.dev`, search public profiles, compare, shortlist, and open the interview kit.
6. Sign in as `college@skillproof.dev`, show the demo cohort, skill gaps, placement readiness, and employer share.
7. Sign in as `admin@skillproof.dev`, show provider health, run trace, evidence, skill runs, prompts, and audit logs.
8. Return to candidate and start `/candidate/new-verification` for a live repository path.

## Talk Track

SkillProof is not a resume scorer. It verifies what a developer can build, explain, test, debug, and maintain using repo evidence, terminal proof, ownership verification, own-code interviews, AI-collaboration challenges, and validator-audited score contracts.
